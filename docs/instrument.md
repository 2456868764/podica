# 什么是Podcast
##  多源内容上传
  - 上传文件
  - 上传网站
  - 文本

##  智能播客生成器 (Podcast Generator) 
- 基于AI的内容分析与结构化
- 自动生成播客剧本和对话
- 多角色语音对话生成
- 智能内容摘要与重点提取
 ## Postcast 模版
  - 可创建、编辑、删除模板
  - 模板包含以下自定义选项：
    - 播客名称和标语
    - 特定内容指令
    - 发言人角色和语音语调
    - 多语言支持
    - 对话风格和互动技巧
    - 对话结构设计
    - 单集时长控制
    - 语音模型选择
    - 背景音乐和音效集成
    - 开场和结尾模板定制

内容基本要求（文本）：

（1）内容表述清晰易懂，口语化，符合播客文本剧本的形式和内容要求;
（2）N个角色互动应高度拟人，具有真实的对话方式（如：对话设计上应足够精巧，通过感叹词、自然反应和话题流动来反映自然的人类对话）;
（3）文本内容质量上，应做到基本信息准确合理，内容质量高有深度有价值，原文内容的全面呈现、升华内容的丰富有趣。
内容基本要求（音色）：
（1）音频要符合播客的基本要求，有开场、结尾、主题，有人声的拟声感、语气、语调、音色、口癖、微情绪、人设感知的统一性、口吻；

（2）具体的音色、语音语调应高度拟人化，媲美真人，有生动丰富的情感传达和自然情绪波动的音调，而不是听起来机械生硬或平淡无情绪；


（3）相关使用音效、音乐、音乐与人声协调；音频融入舒适的环境音效，营造播客录音室的感觉。
